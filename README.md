# numguess

사용자로부터 이름과 정답을 얻고자 합니다.

## Installation

```shell
$ pip install
```

## How to start

```shell
$ python numguess.py
```

## Features

- Get user name
- Greet user with that name
- Get user's guess
- Create random answer number(1-100)
- Compare answer with user's guess
- Print user's guess
